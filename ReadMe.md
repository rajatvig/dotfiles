# Bootstrap for Mac OSX Mojave

### Prerequisites
1. Xcode
2. Brew

### Getting bootstrapped

```
git clone git@github.com:rajatvig/dotfiles.git ~/dotfiles
cd dotfiles
make bootstrap
make osx
```

Other targets are defined when `make` is run without a target

```bash
make
```
